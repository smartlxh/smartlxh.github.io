观察者模式

Posted on 2017-01-31

定义

定义了对象之间的一对多依赖，这样一来，当一个对象改变状态改变时，它的所有依赖者都会收到通知并自动更新

示例

//被观察者接口
public interface Subject{
    //实现类中持有Observer对象的collection，当状态有变化时，通过调用observer的update方法通知更新
    public void registerObserver(Observer o);

    public void removeObserver(Observer o);
    public void notifyObservers();
}


//观察者接口
public interface Observer {
    public void update(args[]);
}

JAVA内置观察者模式

java.util.Obserable类 对应上面的Subject接口

Observable:

addObserver()
deleteObserver()
notifyObservers()
setChanged()

java.util.Observer接口 对应上面的Observer接口

Observer:
update()

可观察者送出通知：
先调用setChanged（）方法，标记状态已经改变的事实
然后调用notifyObservers（）或notifyObservers（Object arg）

观察者接受通知：
实现了update（Observable o,object org）方法
